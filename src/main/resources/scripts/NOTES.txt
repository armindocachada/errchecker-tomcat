Tomcat Error Checking Notes

PREREQUISITES:
I used the pyinotify library. Please install with pip3 install pyinotify. This library helps a lot in the work of detecting new files or modifications
to existing files


1. My understanding is that we only want the errors count during the startup. Errors that happen during the server run after startup are still logged,
to the /tmp/errchecker-log.txt file, but not counted. We detect the server start by using the string "Using CATALINA_BASE".
2. I have used very simple  regular expressions and limited myself to the ones requested in the assignment.
3. The script handles files being renamed,deleted inside the log directory as you might expect from log rotation. It doesn't lose track of the fles.
4. The script only parses files/lines that are created and modified after it is running in the background. When it runs for instance, if it finds a 2GB log file
it will only start parsing new lines added to it.
5. For the script to work we make of the file size and use seek() method to start reading from last known positionj in the file


------- RUNNING THE ERRCHECKER script ------------------------

In order to make it easy for myself and for the person reviewing the errchecker script, I created a small docker container that runs a tomcat server and
the errchecker script inside the container. The script also builds the application using maven also by using a readily available mvn docker image.

Instructions for MACOS X

Assuming you have docker installed in your machine, inside the errchecker-source directory execute the following:

$ /bin/bash run.sh

It might take a while as docker will have to download the tomcat and mvn images from dockerhub

once all finished you can access tomcat through port 8080

if you wish to look inside the container you can go inside the container using docker exec -it <container id>

You will be able to see that the tomcat logs are already being redirected to /tmp/tomcat-logs and you will see an entry for the error check logs
in /tmp/errchecker-log.txt
